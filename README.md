# SportCompare ⚡

Comparateur de prix sport avec Score IA intégré.

## Structure du projet

```
sportcompare/
├── index.html          ← Point d'entrée HTML
├── vite.config.js      ← Config Vite
├── package.json        ← Dépendances
├── vercel.json         ← Config Vercel
└── src/
    ├── main.jsx        ← Point d'entrée React
    ├── App.jsx         ← Application principale ← À MODIFIER
    └── index.css       ← Reset CSS
```

## Déploiement

1. Pushez ce dossier sur GitHub
2. Importez le repo sur Vercel
3. Vercel détecte Vite automatiquement → Deploy

## Développement local (optionnel)

```bash
npm install
npm run dev
```
